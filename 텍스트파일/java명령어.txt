기초 배경 지식

 초기화 : 변수를 선언하고 변수에 값을 넣어주지 않았을 때 '초기화 하지 않았다.'라고 표현함
          : 변수에 최초로 값을 대입하는 행위를 변수 초기화라고 한다. 이때 값을 초기값이라 한다.

 리터널 : 코드에서 프로그래머가 직접 입력한 값을 리터널이라고 부른다.
